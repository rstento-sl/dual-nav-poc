SnapLogic Documentation POC - Dual Navigation Demo
===================================================

HOW TO VIEW:
1. Extract this zip file
2. Open "index.html" in your web browser
3. Explore the dual navigation:
   - Click "All Products" to browse by product (Admin Manager, AutoSync)
   - Click "All Tasks" to browse by workflow (Getting Started, Configuration, Monitoring)

FEATURES:
- Product-based navigation with filtering
- Task-based navigation organized by workflow
- 27 topics reused across both views (single source of truth)
- Modern branded UI with SnapLogic colors
- Responsive design (mobile and desktop)

NAVIGATION:
- Homepage: index.html
- Product View: product/admin-manager/ or product/autosync/
- Task View: tasks/admin-manager/ or tasks/autosync/

TECHNICAL DETAILS:
- Built with DITA 1.3 and keyref architecture
- 202 source topics, 229 total references
- Content reuse without duplication
